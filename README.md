# 🍰 Delícias do Ryan

Landing Page desenvolvida com foco em responsividade, organização de código e boas práticas de desenvolvimento Front-End.

Este projeto simula o site de uma confeitaria, com layout moderno e adaptável para diferentes tamanhos de tela.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- Media Queries
- JavaScript (menu hambúrguer)

---

## 📱 Responsividade

O projeto foi desenvolvido com foco em:

- Desktop
- Tablet
- Mobile
- Telas menores que 480px

---

## 🎯 Funcionalidades

✔ Menu responsivo com hambúrguer  
✔ Layout adaptável  
✔ Seção de sabores  
✔ Seção sobre  
✔ Contato com mapa
